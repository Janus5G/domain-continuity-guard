# Roadmap

## Near term

- Add a resolver interface instead of direct `dns_get_record()` calls.
- Add PostgreSQL schema.
- Add WebAuthn/passkey integration examples.
- Add recovery-code implementation with hashed codes.
- Add a Laravel service provider.
- Add Symfony bundle example.

## Medium term

- Add RDAP/registrar expiry monitoring examples.
- Add Certificate Transparency monitoring examples.
- Add webhook events for epoch change, continuity failure and recovery block.
- Add hosted-provider reference UI.
- Add migration tooling.

## Long term

- Draft an interoperable Domain Continuity Identity specification.
- Publish test vectors for continuity signatures.
- Build integrations for webmail/control-panel platforms.
